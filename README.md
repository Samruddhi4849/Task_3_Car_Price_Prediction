# Task_3_Car_Price_Prediction
## Objective
To predict the selling price of used cars using machine learning techniques.

## Dataset
Used Car Price dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Used
Linear Regression

## Conclusion
The model predicts car prices based on important features such as year, mileage, fuel type, and transmission.
